# CLAUDE.md

Instructions for Claude Code in this repo.

- Read AGENTS.md: it contains the project invariants and the integration
  and validation procedures. All of it applies.
- Read CONTRIBUTING.md for the architecture (SDK / hub / dashboard) and the
  commit rules (Conventional Commits, checked by commitlint).
- Validation before concluding any task:
  `npm run typecheck && npm test && npm run build`, plus the `node --check`
  of the script extracted from server/dashboard.html if the dashboard changed,
  plus a hub smoke test (`RN_DEVTOOLS_TOKEN=dev bun server/server.mjs` then
  curl the dashboard and the /mcp endpoint) if the server changed.
- Never add a dependency to the SDK (src/client) nor a build step to the
  dashboard. Native or system features are always optional and degrade
  cleanly.
- CHANGELOG.md is generated by release-it: do not edit it by hand.
