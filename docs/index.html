rn-devtools-hub

Local inspector for React Native and Expo apps. Network, crashes, logs, storage, SQLite, cache, screen mirror, remote actions and an MCP server, streamed to a web dashboard. Zero native dependencies, inert in production, your data stays on your machine.

GitHub Integration guide Protocol (reference) llms.txt

Getting started in 3 steps

# 1. In your project (devDependency)
npm install --save-dev rn-devtools-hub

# 2. The hub, from the project root (Bun required)
npx rn-devtools-hub

# 3. In the app: 3 lines of glue, see the integration guide
