            Control the phone without a cable (adb over Wi-Fi)
            

              adb over Wi-Fi enables the mirror with tap and keys without a USB cable.
              On the phone:

              1. Settings, About phone: tap "Build number" 7 times

              2. Settings, Developer options: enable "Wireless debugging"

              3. Open "Wireless debugging", choose "Pair device with pairing code"

              4. Enter here the ip:port address and the 6-digit code shown:
            

            

              
              
            

            
            

              5. Then connect using the ip:port address shown on the MAIN
              "Wireless debugging" screen (different port from pairing):
            

            
            
            

              Alternative with a cable: plug in once via USB, run
              "adb tcpip 5555" in a terminal, unplug, then connect here
              with phone-ip:5555. Pairing is not required in that case.
